# AI Travel Planner

An AI-powered travel planning web application built using Python and Groq LLM API.

## Features

- Generates travel itineraries
- Takes:
  - Destination
  - Budget
  - Number of people
  - Number of days
  - Interests
- Uses Groq LLM API
- Web-based interface

## Tech Stack

- Python
- Flask
- HTML/CSS
- Groq API

## Setup

pip install -r requirements.txt

python app.py
