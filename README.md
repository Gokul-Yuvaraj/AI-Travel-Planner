## Features

- Generates travel itineraries
- Takes:
  - Destination
  - Budget
  - Number of people
  - Number of days
  - Interests
- Uses Groq LLM API
- Web-based interface
- Interest-based recommendations
- Multi-day travel schedules

## Tech Stack
- Python
- Flask
- Groq API
- HTML/CSS
- JavaScript
- Git & GitHub
- Render
